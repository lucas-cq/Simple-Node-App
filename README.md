## Give the Terminal a Beer!
Give the terminal a type of beer and an amount.

### Commands
```
--beer=IPA --amount=1
```

```
--beer=Lite --amount=2
```

```
--beer=Stout --amount=3
```

### Third Party Integration
* I used [Yargs](https://www.npmjs.com/package/yargs) to make my own arguments. Line 20 as an example
